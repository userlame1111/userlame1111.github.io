Десктопное приложение на Flutter. Тёмно‑зелёная тема, удобные карточки, асинхронная работа с Supabase и доступ к данным только для владельца.

Приватность

Записи из Supabase видит только владелец. RLS‑политики обеспечивают доступ “только к своим данным”.

Комфорт

Тёмно‑зелёная тема, адаптивные панели, аккуратные карточки с индикатором стойкости пароля.

Авторизация

Вход по email/логину, подтверждение почты, восстановление пароля по почте или логину.

Как использовать подтверждение e‑mail

  1. Зарегистрируйтесь в приложении и проверьте почту.
  2. Перейдите по ссылке из письма — вы окажетесь на этой странице.
  3. После подтверждения выполните вход в приложение.

Если письмо не пришло — проверьте спам или запросите повторную отправку.
